2022-11-01

今天学会了如何在GitHub Pages上面搭建自己的博客，记录一下吧，免得又忘记，然后就是CSDN，博客园大逛。

官网：https://git-scm.com/

安装Git好之后，直接桌面右键，可以找到Git Bash Here，点击打开

4.配置SSH Key

注意，这一步的前提是你已经拥有了GitHub账号，没有的话，注册一个

使用SSH的目的是可以免密的将本地的源码和资源上传到github，无需要每次都输账号和密码，并且这样也更安全。

首先，生成SSH Key，输入：ssh-keygen -t rsa -C “邮件地址”，这个邮件地址是无所谓的，随便写一个

输入之后，连续敲三次回车，就会在你的用户文件夹中生成.ssh文件夹

把.pub中的内容复制

来到GitHub官网，你的主页，点击SSH and GPG keys，点击New SSH Key

填写title和SSH Key

点击Add SSH Key

测试是否连接上：ssh -T git@github.com ,如下就为成功

配置用户名和邮箱

git config –global user.name “你的用户名”

git config –global user.email “你的邮箱”

5.搭建博客

hexo的基本命令

更多的可以参考官网：Hexo

新建一个保存博客项目的地址：比如我的 E:\ma-yuan-blog

在终端进入这个目录

cd E:\ma-yuan-blog

初始化：hexo init

初始化后的博客文件夹

生成静态网页：hexo g

本地启动：hexo s

访问 http://localhost:4000/

6.部署到GitHub

新建仓库并且要以此格式命名：用户名.github.io

勾选README,创建

编辑_config.yml

发布：hexo d

如果遇到这个错误

Deployer not found: git

那就安装这个依赖：npm install hexo-deployer-git –save

访问 http://ma-yuan-github.github.io/
